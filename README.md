# Peach

## Migrations

`cd Database`

`export FAUNADB_SECRET=YourFaunaSecretKey`

`npx migrate`

## Dev

`export FAUNADB_SECRET=YourFaunaSecretKey`

`export JWT_SECRET=YourJWTSecretKey`

`npm i`

`netlify dev`

`open http://192.168.1.108:8888/ in browser`

## Deploy

`expo build:web`

`netlify deploy`
